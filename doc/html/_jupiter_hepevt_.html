/jupiter/hepevt/

Commands to control HEPEvt event interface.

Sub-directories :

Commands :


file [FileName]

Set a file name of HEPEvt data

Available at all Geant4 states.

Parameters
FileName type s Omittable : default value = pythia_event.data


numberOfSkipEvents [N]

Set number of skip events.

Range : N>-1

Available at all Geant4 states.

Parameters
N type i Omittable : current value is used as the default value.
