/jupiter/

Jupiter control commands.

Sub-directories :


/jupiter/hepevt/

Commands to control HEPEvt event interface.


/jupiter/cain/

Commands to control CAIN event interface.


/jupiter/beam/

Particle Beam control commands.


/jupiter/run/

Commands to control Jupiter run information.


/jupiter/tracking/

JUPITER Tracking commands.

Commands :


generator [generator]

Select primary generator.
Available generators : HEPEvt, CAINEvt, ParticleGun, ParticleBeam

Available at all Geant4 states.

Parameters
generator type s Omittable : default value = ParticleGun Parameter candidates : HEPEvt CAINEvt ParticleGun ParticleBeam
