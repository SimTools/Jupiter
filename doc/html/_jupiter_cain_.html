/jupiter/cain/

Commands to control CAIN event interface.

Sub-directories :

Commands :


file [FileName]

Set a file name of CAIN data

Available at all Geant4 states.

Parameters
FileName type s Omittable : default value = cain.dat


gengamma [GenGamma]

Gamma track is passed to Jupiter

Available at all Geant4 states.

Parameters
GenGamma type b Omittable : default value = 1


genelectron [GenElectron]

Electron track is passed to Jupiter

Available at all Geant4 states.

Parameters
GenElectron type b Omittable : default value = 1


genpositron [GenPositron]

Positron prack is passed to Jupiter

Available at all Geant4 states.

Parameters
GenPositron type b Omittable : default value = 1


gendirection [GenPositron]

Only particles generated in the given direction is passed to Jupiter
1.0(+Z direction), -1(-Z direction), 0(+-Z)

Available at all Geant4 states.

Parameters
GenPositron type d Omittable : default value = 1


verbose [Verbose]

Verbose output

Available at all Geant4 states.

Parameters
Verbose type i Omittable : default value = 0
