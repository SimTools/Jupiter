beamtype [BeamType]

Set beamtype. Type 0 or 1.
0 : isotropic, 1 : gaussian

Range : BeamType>=0

Available at all Geant4 states.

Parameters
BeamType type i Omittable : current value is used as the default value.


convergence [IsConvergenceMode]

Set beam convergence mode. Type 0 or 1.
0 : divergence beam
1 : injected from surrounding sphare to beam position

Available at all Geant4 states.

Parameters
IsConvergenceMode type b Omittable : default value = 0


flatpos [IsFlatProfile]

Set beam profile type. 0 or 1.
0 : Gaussian profile
1 : Flat profile

Available at all Geant4 states.

Parameters
IsFlatProfile type b Omittable : current value is used as the default value.
